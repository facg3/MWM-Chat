 ### Run this project :laughing

 - Clone the repo:

 SSH 
 ```
   git@github.com:facg3/MWM-Shop_Project.git
```

 Http 
 ```
   https://github.com/facg3/MWM-Shop_Project.git
```

 - Install the node.js devDependencies:
```
 npm install
```
 - Run the (module) Tests
```
 npm test
```
 - Run the Server with Nodemon
```
 npm run watch
```
